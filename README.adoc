Preparar:

[source,shell]
----
go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest
go install github.com/favadi/protoc-go-inject-tag@latest
----

Gerar protocolo:

[source,shell]
----
protoc --go_out=. --go_opt=default_api_level=API_OPAQUE --go_opt=paths=source_relative --go-grpc_out=. --go-grpc_opt=paths=source_relative protocol/protocol.proto
----
